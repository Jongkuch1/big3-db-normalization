# big3-db-normalization

This repository contains the source data and normalization deliverables for the Big3 construction database project.

## Contents

- `big3_construction_raw_data.csv` - raw denormalized input dataset
- `normalization_submission/1NF/` - first normal form outputs
- `normalization_submission/2NF/` - second normal form outputs
- `normalization_submission/3NF/` - third normal form outputs
- `normalization_submission/4NF/` - fourth normal form outputs
- `normalization_submission/BCNF/` - Boyce-Codd normal form outputs

## Submission Files

The normalized folders include CSVs for the core entities in each form, such as projects, workers, suppliers, equipment, materials, certifications, and skills.

The repository also includes the written normalization explanation in `normalization_submission/normalization_explanation.pdf`.
